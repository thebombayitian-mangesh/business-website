# Modern Business Website

A modern responsive business website built using React.js, CSS3, and JavaScript ES6+.

## Features

- Responsive Design
- React Router Navigation
- Dark Mode Toggle
- Mobile Navbar
- Contact Form Validation
- CSS Grid and Flexbox
- Interactive UI Components

## Technologies Used

- React.js
- JavaScript ES6+
- CSS3
- React Router DOM
- Vite

## Installation

```bash
npm install
npm run dev
```

## Pages

- Home
- About
- Services
- Portfolio
- Contact
